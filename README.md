AI enabled IOT-Based AQI Monitoring Device
 A smart, portable IoT system that:
 Continuously monitors air pollutants (CO₂, CO, O₃, VOCs, PM2.5).
 Sends real-time data to a cloud platform via wireless connectivity.
 Provides users with live AQI insights and AI-based predictions.
 Enables authorities and individuals to take proactive actions against pollution
